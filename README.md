# HealthHub OS

A unified operating system for managing multiple digital health companies from a single command center.

## Overview

Built for founders and investors running portfolio health ventures. Replaces fragmented systems with one coherent platform providing:

- **Portfolio Dashboard** — Cross-company KPIs, revenue tracking, and health scores
- **Company Switcher** — Instant context switching between companies
- **Unified Patient Flow** — Shared patient database with company-level segmentation
- **Financial Command Center** — Consolidated economics with investor-ready reporting
- **Staff Management** — Shared resource pool across companies
- **Real-time Activity** — Live feed of events across all companies

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Recharts
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Configured for Vercel deployment. Push to `main` to trigger auto-deploy.
